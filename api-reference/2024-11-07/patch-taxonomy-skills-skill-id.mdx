---
title: 'Update Company-Specific Skill'
openapi: 'PATCH /taxonomy/skills/{skill_id}'
---

Submit the most up-to-date Company-Specific Skill information. Any field that is present will overwrite existing values within the system, while absent fields will be left as is.<br/> **Please note**: You don't have to update TechWolf skills, the latest vocabulary is always present in your tenant.
